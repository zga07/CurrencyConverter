# CurrencyConverter
Простой конвертер валют, работающий с API exchangerate-api.com
Чтобы использовать этот конвертер, вам нужно получить API ключ на сайте exchangerate-api.com и сохранить его в переменной окружения EXCHANGE_API_KEY. (export EXCHANGE_API_KEY="ваш ключ")
Далее запустите приложение с помощью команды "go run main.go".
